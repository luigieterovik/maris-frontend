# Mari's Boutik's Frontend

[Mari's Boutik's](https://maris-frontend.vercel.app/) é o site oficial para compra dos produtos cosméticos fabricados por Mari's Boutik's. O usuário poderá registrar uma conta e realizar compras dos produtos disponíveis. *(Em desenvolvimento)*

<img src="https://i.imgur.com/PoUppkT.png">

## Tecnologias Utilizadas

- React
- Styled-components
- Axios
- React Router

## Requisitos

- Node.js (versão 16 ou superior)
- npm ou yarn

## Instalação

1. Clone o repositório:
`git clone https://github.com/luigieterovik/maris-frontend.git`

2. Acesse a pasta do projeto:
`cd maris-frontend`

3. Instale as dependências:
`npm install`

## Execução

Para iniciar o projeto em ambiente de desenvolvimento, execute:
`npm start`

O site estará disponível em `http://localhost:3000`.

## Estrutura do Projeto

- `src/`: Contém o código principal;
  - `components/`: Componentes reutilizáveis;
  - `Containers/`: Páginas do site;
  - `assets/`: Imagens do site;
  - `styles/`: Estilos globais;
  - `contexts/`: Contextos do projeto;
  - `services/`: Configurações de APIs;
  - `routes/`: Configuração de todas as rotas;

## API

Este projeto consome uma API desenvolvida separadamente. Você pode acessar o repositório do backend no link abaixo:

[Backend da API - Mari's Boutik's](https://github.com/luigieterovik/maris-backend)
